项目经历

在iOS移动端的项目开发中，完整的参与过两个商业项目，单独做的有一个私人商业项目，一些个人小项目，还有一些是工作早期参与的项目。在Web端，有基于Hexo驱动搭建于Github的个人定制博客，以及这个具有响应式布局能够适配移动端、PC端的个人简历。

易贷e禄 (15.11-16.03)

易贷e禄是一个私募销售管理平台，提供丰富的理财产品。具有用户注册、实名认证、充值、购买、提现（退款）以及平台的广告、不同的产品不同的付息方式、不同收益率、募集期设置、募集期收益、流标收益等相关的核心P2X的功能。

我们开发的是这个产品的iOS移动端App，里面集成了第三方银联支付功能以及为确保用户信息安全的同卡进出机制。

我主要负责协助另外一个小组对项目中的用户对理财产品的购买流程进行实现，并且使用HTML对与产品无关更多部分进行便捷开发。

找找 -- 宠卫宝 (14.11-15.10)

找找是一款主打智能定位功能的物联网产品，主要针对于目前老人走失、幼童被拐、宠物走丢等热点社会问题，拥有独创的安全范围以及智能警报功能，对于定位的功能实现提供两种方案，根据硬件所处位置信号强弱在LBS和GPS之间进行智能切换。

软件拥有地图相关方面的范围越界警告、经纬度纠偏，集成融云SDK完成社交方面的功能需求。

我主要负责的是产品从1.0到2.0的iOS端研发，使用Objective-C开发，期间使用HTML5进行过一小部分的便捷开发。在宠卫宝方面有专门针对于宠物健康问题的一系列功能，提供有专门的宠物医生为宠物进行日常饮食以及健康方面的资讯。

梦梦(Sliip) (14.12)

“梦梦”是一款致力于提高睡眠质量的智能家居物联网产品，通过智能枕头进行睡眠以及周边环境的数据进行信息采集，经过独有算法通过移动端设备反馈给用户准确的睡眠数据。

我负责的是项目前期的基础功能实现，涉及到的有通过Core Motion中的加速器以及陀螺仪对用户睡眠数据的获取，对收集到的数据进行优化处理，并将处理过后的数据以表格、图表的形式展示给用户。

主要在Kickstarter上进行前期的众筹，但是由于项目在筹备资金阶段没有达成目标，项目没有启动，现在已经被终止。

韩流圈 (14.07-14.10)

韩流圈是一款为国内韩流粉丝提供明星动态、韩国节日资讯、热门影视、境外旅游等服务的资讯类App。

小组主要进行软件的初代版本开发，为软件接入第三方SDK，实现语音留言功能，实现基本的新闻资讯展示功能。

Run Coach 5K (14.06)

Run Coach是一款引导类的运动App，主要适用的是运动能力有限并且意志力薄弱人群，通过一系列的跑步教程循序渐进的帮助用户在短期内达到能够跑步5公里的目的。

项目中使用GPS对用户进行运动情况的获取，并且根据选取的教程对用户的运动量进行阶段性的提示，最后在运动过后使用地图为用户展示当次的运动情况。

由于接触到的时候项目已经快接近尾声，并且当时自己水平有限，仅仅对项目中的由于iPhone迭代照成的屏幕适配问题进行解决。

在搭建博客前期分析比较过Hexo、Ghost、Jekyll等静态博客框架，当时Hexo正值热门，Ghost的管理后台操作起来比较方便，Jekyll的社区比较成熟，最后在权衡生成速度、部署简便与国人开发之后使用了Hexo搭建了个人博客。
