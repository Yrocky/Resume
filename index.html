易贷e禄 (2015.11-2016.03)

易贷e禄是一个P2P产品的iOS客户端。

该软件使用连连支付集成银联支付功能，支持主流银行的支付，所有涉及到金钱以及密码的输入均使用特制安全键盘并且对输入内容进行MD5加密。开发中的布局全部使用Masonry进行Auto Layout。

项目整体是MVC的架构，使用AFNetworking2.X版本进行网络请求的发起，与功能无关的帮助中心界面使用HTML5结合原生进行便捷开发。

接入友盟进行用户行为的统计，并且使用Share SDK集成主流社交平台的分享功能。

找找 -- 宠卫宝 (2014.11-2015.10)

找找是一款主打智能定位功能的物联网产品，主要针对于目前老人走失、幼童被拐、宠物走丢等热点社会问题，拥有独创的安全范围以及智能警报功能，对于定位的功能实现提供两种方案，根据硬件所处位置信号强弱在LBS和GPS之间进行智能切换。

软件使用Map Kit和Core Location框架实现用户定位、范围越界警告等功能，在客户端有针对于高德地图的经纬度纠偏，集成融云SDK完成社交方面的功能需求。

使用ZBar完成硬件设备和客户端之间的扫描绑定功能，使用HTML5进行一小部分的用户问卷开发。在宠卫宝方面有使用TEAChart图表进行宠物的运动信息展示，并且使用建模从运动量以及体征给用户提供出宠物的健康状况。

OurFind (2015.10-至今)

OurFind是一款快递查询软件，具有查询快递信息以及历史快递信息查询等功能，从软件设计到以及开发实现是我与朋友一起完成的。

一直有在进行版本迭代，目前适配iOS9+，拥有3D Touch功能。

OurFind使用ZBar提供条形码扫描功能，能够有效正确的识别物流信息。使用FMDB实现数据库方面的功能，数据库用于对物流信息的存储，并且方便用户查询快递。

OurDay (2015.10-至今)

OurDay是一款用于倒数日提醒的应用，和OurFind一样从软件设计到以及开发实现都是我与朋友一起完成的。

OurDay通过使用通知推送实现倒计日提醒功能，使用FMDB进行存储用户添加的事件，使用iCarousel库实现用户的倒计日四种分类事件，使用Pop进行主要动效的实现。

另外使用Foundation下的NSCalendar类实现一些日期计算功能。使用Share SDK结合微信开放平台和微博开放平台实现第三方分享功能。

梦梦(Sliip) (2014.12)

“梦梦”是一款致力于提高睡眠质量的智能家居物联网产品，通过智能枕头进行睡眠以及周边环境的数据进行信息采集，经过独有算法通过移动端设备反馈给用户准确的睡眠数据。

项目中使用Core Motion框架中的加速器以及陀螺仪对用户睡眠数据的获取，对收集到的数据在服务器端优化，客户端使用FMDB进行数据库存储，并将处理过后的数据使用RPRadarChart以表格、图表的形式展示给用户。

使用Share SDK进行主流社交平台的分享功能实现，接入融云SDK使软件拥有社交功能，方便用户之间的睡眠质量之间的比较。

韩流圈 (2014.07-2014.10)

韩流圈是一款为国内韩流粉丝提供明星动态、韩国节日资讯、热门影视、境外旅游等服务的资讯类App。

软件使用Share SDK实现第三方登录功能、社交平台分享等功能。使用Core Text进行新闻资讯的展示，使用AVFoundation框架实现对资讯的用户语音评论功能。

Run Coach 5K (2014.06)

Run Coach是一款引导类的运动App，主要适用的是运动能力有限并且意志力薄弱人群，通过一系列的跑步教程循序渐进的帮助用户在短期内达到能够跑步5公里的目的。

使用Core Location框架进行用户运动情况的获取，结合Map Kit进行展示用户运动轨迹，使用Core Data进行用户运动数据的存储。使用Core Animation框架实现用户运动的时候的倒计时沙漏动画效果。

同时解决由于手机版本迭代产生的屏幕适配问题，以及由于SDK升级造成的定位功能之间的版本差异。

个人简历 (2016.03)

这是一个使用HTML+CSS以及少许的JavaScript编写的静态个人简历站点，部署在Github上。从界面设计到具体实现都是我一个人完成的，拥有响应式布局功能，能够完美适配移动端的iPhone6+、iPad以及PC端的主流浏览器。

使用Sublime Text2编辑器进行开发，调试使用的是Chrome浏览器、iPhone6及以上、iPad mini等设备。具有流畅的动画效果，主要是使用JavaScript结合DOM进行实现，响应式布局方面使用的是CSS3中新增加的选择器以及媒体查询功能。
